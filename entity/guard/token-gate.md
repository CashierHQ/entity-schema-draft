### Token Gate

#### Props

- `address`: Address of the collection
- `chain`: Chain to which the collection belongs
- `amount` : minimum balance of user

#### Example

```json
{
  "tokenGate": {
    "address": "0x4B4c8ABe2dC873F04068075908F4719554b2cb2C",
    "chain": "ETH",
    "amount": 300
  }
}
```
