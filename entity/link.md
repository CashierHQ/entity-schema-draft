### Link

| Property | Type        | Description                         | Value          |
| -------- | ----------- | ----------------------------------- | -------------- |

| type     | String      | Type of the link (airdrop, invoice) | Airdrop        |
| standard | String      | Type of token for airdrop           | ICRC1          |
| guard    | Array Guard | Array of guard                      | [Guard]        |
